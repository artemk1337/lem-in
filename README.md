# Lem-in

## Quick start

1. Download:
    ```bash
    git clone https://github.com/artemk1337/lem-in.git
    ```
2. Install:
    ```bash
    make
    ```
3. Run:
    ```bash
    ./generator <flag> | ./lem-in
	```
	or  
	```bash
	./generator <flag> | ./lem-in -v
    ```
	or  
	```bash
	./generator <flag> | ./lem-in | python3 visualizer.py
    ```
See `./generator --help` for usage flags.
